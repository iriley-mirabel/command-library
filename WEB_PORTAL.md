# Web Portal - Coming Soon

The Next.js web portal for browsing and searching commands is planned for a future phase.

## Planned Features

Based on the PRD, the web portal will include:

### Pages
- **Home (/)** - Dashboard with "Most Used" commands and search
- **Library (/library)** - Grid/list view of all 75+ commands
- **Command Detail (/command/[id])** - Individual command pages with copy functionality
- **Install (/install)** - Installation guide

### Features
- Search and filter commands by name, tag, or category
- Copy command markdown to clipboard
- Download all commands as a .zip file
- Dark mode theme (Slate/Zinc)
- Developer-focused aesthetic

### Technical Stack
- Next.js (App Router)
- Tailwind CSS
- Shadcn UI

## Current Status

The command library is fully functional via:
- Direct file access in `.cursor/commands/`
- Setup scripts for integration
- Documentation in README.md

The web portal will be built in a future phase to provide a more user-friendly interface for browsing and discovering commands.

## Contributing

If you'd like to help build the web portal, see the PRD in `prd.txt` for detailed specifications.
