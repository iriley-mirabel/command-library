# /fix-spacing-layout

Convert hardcoded spacing to design tokens, fix layout inconsistencies.

## Purpose
Convert hardcoded spacing to design tokens, fix layout inconsistencies

## Usage
`/fix-spacing-layout [path]`

## Speed
Fast (30s - 2min)

## When to use
Design token compliance, layout standardization

## Fixes
- Hardcoded px values → Design tokens
- Inconsistent spacing → Standard spacing scale
- Inline style spacing → Tailwind classes
- Layout alignment issues
