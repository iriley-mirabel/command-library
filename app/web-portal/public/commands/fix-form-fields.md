# /fix-form-fields

Standardize form fields using the project's Label components and spacing rules.

## Purpose
Standardize form fields with Label component, proper spacing

## Usage
`/fix-form-fields [path]`

## Speed
Fast (30s - 2min)

## When to use
Form inconsistencies, design token compliance

## Instructions
1. **Components**: Replace plain HTML labels with the standardized `Label` component
2. **Text**: Standardize placeholder text for consistency across the module
3. **Logic**: Add missing validation logic and error state handling
4. **Spacing**: Fix field spacing and ensure design token usage (hardcoded px to tokens)

## Usage
- Use for form inconsistencies and design system compliance

## Fixes
- Replaces plain labels with Label component
- Standardizes placeholder text
- Adds missing validation
- Fixes field spacing
- Ensures design token usage
