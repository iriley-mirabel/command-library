# /test-page-quick

Perform a rapid smoke test of the current page. Identify rendering errors, broken API calls, or console errors. Verify critical paths like Dashboard, Products, and Customers.

## Purpose
Quick smoke test of critical pages

## Usage
`/test-page-quick`

## Speed
Fast (2min)

## When to use
Daily checks, quick validation

## Tests
- Dashboard
- Products
- Chargebrite Customers
- Chargebrite Subscriptions
- Classified Ads
- Events
