# /implement-view-modes

Add missing Table and Card view toggles to the current page.

## Purpose
Add missing table/card views to pages

## Usage
`/implement-view-modes [path]`

## Speed
Moderate (2-4min)

## When to use
Pages with only one view, missing view implementations

## Instructions
1. **Views**: Implement both a standardized Table view and a responsive Card view
2. **State**: Add the necessary React state management to handle view switching
3. **UI**: Add the view toggle component to the existing Filter Bar

## Usage
- Use when a page only has one view type but requires both

## Fixes
- Adds missing table view
- Adds missing card view
- Adds view state management
- Adds view toggle to filter bar
