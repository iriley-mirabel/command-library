# /find-command [keyword]

Search the command library to find the right tool for the task.

## Purpose
Search commands by keyword, category, or purpose

## Usage
`/find-command [keyword]`

## Speed
Fast (30s)

## When to use
Need to find a command, discover commands

## Usage Guide
Search by these categories:
- **"UI"**: `/fix-filter-bar`, `/fix-data-table`, `/fix-form-fields`
- **"Testing"**: `/test-page-quick`, `/debug-failing-page`
- **"Cleanup"**: `/clean`, `/imports`, `/refactor`

## Scenarios
- **First Day?** Run `/test-page-quick`
- **Before PR?** Run `/pre-commit-checklist`
- **Fixed a bug?** Run `/daily-cleanup`

## Searches by
- Keyword (e.g., "filter", "test", "cleanup")
- Category (e.g., "UI", "testing", "cleanup")
- Purpose (e.g., "fix imports", "remove dead code")

## Common Searches
- **"fix imports"** → `/fix-import-paths`
- **"fix spacing"** → `/fix-spacing-layout`
- **"fix filter bar"** → `/fix-filter-bar`
- **"test page"** → `/test-page-quick` or `/test-pages`
- **"cleanup code"** → `/cleanup-unused-code`
