# /command-usage-report

Generate comprehensive usage report for all slash commands.

## Purpose
Generate comprehensive usage report for all slash commands

## Usage
`/command-usage-report`

## Speed
Fast (30s - 1min)

## When to use
Weekly/monthly analysis, identify popular commands

## Generates
- Top 10 most used commands
- Unused commands (candidates for removal)
- Usage trends and patterns
- File associations
- Recommendations
